# 🕉️ Spiritual Mantra Audio Generator

This notebook converts Sanskrit mantras into devotional MP3 audio using Python.

## ✨ Features
- Convert mantra text to speech
- Generate separate `.mp3` files
- Female devotional voice effect
- Audio enhancement using `pydub`
- Easy mantra customization

## 🛠️ Technologies
- Python
- gTTS
- pydub
- FFmpeg
- Google Colab

## 📌 How to Use
1. Open the notebook in Google Colab
2. Install required libraries
3. Add your mantras inside the `MANTRAS` dictionary
4. Run all cells
5. Download generated MP3 files

## 🎵 Output
Each mantra will be saved as a separate high-quality spiritual audio file.

## 🙏 Purpose
This project is made for creating peaceful and devotional mantra audio in a simple way.
